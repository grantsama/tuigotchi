# Installation
1. Clone this repo
2. cd into "tuigotchi"
3. Run "make install" (You may be prompted to enter your passcode)
4. Done!

## Dependencies
The ncurses-devel package is needed to build from source.

# Uninstall
1. cd into "tuigotchi"
2. Run "make uninstall" (You may be prompted to enter your passcode)
3. Done!
